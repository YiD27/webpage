Project #4 Grass Looping

Description

Using computer loop and noise language to mimic grass wave in wind, and then there are patterns generated.

Design Process

I started writing a Grass class manly made with array of strokes and dots, adding noise-based movement gave them more real-grass gestures. Then I played with the overlaying effect with multiple grass classes together on the canvas, creating a bigger array. At last I added a function of changing the backgroung color with mouse clicking, the color range is several shades of grass green with showing the grass strokes on deeper background and showing only the dots array on tops of each grass on lighter background. For me, this step is exploring the patterns generated in 1D, 2D even 3D visually.

Reflections

Personally I enjoyed this process of mimicing nature through pure machine language, and I felt a sense of peace when seeing this resulted grass wave flowing on my screen, as if facing a concentrated, and created-by-me grasslands.
