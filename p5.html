Project #5 EarPods Tree

Description

What I have been listening to for an afternoon? I recorded the data of the time period, listening music from Spotify, shifting to TikTok, shifting back to Spotify, shutting down all sounds, then turning on TikTok again. I used this data set to create three array and generate tree-like structures growing up in each's listening time period.

Design Process

I started with collecting datas, which is recording the time and content I had listened to on a Saturday afternoon from 1pm to 5pm, and concentrated the time period to a one-minute range. With an interest in the recursive tree pattern, I coded my tree growng according time phrases, and the concept is "when listening, there's a tree growing from the earpods and in my brain". Last I did some color-adjustment based on sources of content and added a timer text to better explain the image.

Reflections

For this project I tried to add a commercial practice based on this coding art, for example, a sony earphone or Airpods on the backgroung layer can create this sense of earpods advertisement visual.
