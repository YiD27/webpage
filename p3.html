Project #3 Computer Clock_ 'what is time?'

Description

Referring to the clocks we see everyday and those time display on desktop, I thought about what time means to computer when working on this code. When my mouse wondered pointlessly on desktop, things must been happening with the machine, dealing with my data input. So I made this experimental 'clock' to show how time is counted with users' interactions and how time is gone and restarted when users click "recload".

Design Process

What is the 'second', 'minute' and 'hour' in my clock?

Reflections
