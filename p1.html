Project #1 Lost in Translation

Description

This is a drawing I made using shape primitives with p5. It is a leopard stuffed toy lost by my friend, and I drawed this based on her description, tranlated using computer language.

Design Process

I started with a 2D body shape I drew in Illustrator and converted its svg code to p5. And I added facial details of eyes, nose and mouse of a leopard. The leopard pattern is an overlaying image on top. Then by playing with blending models, I intended to create a blurry and fluffy style for the toy, symbolizing it as a lost treasure in memory. Last step, adding flurry outline detials with Illustrator again.

Reflections

In this practice, I used a lot help from Illustrator to draw shapes. I also explored blending modes to realize the furry stuffed toy feeling which was a lot more difficult than I expected. Therefore inspired me on how in human language we are able to imagine a lot more behind word but computer is unable.
